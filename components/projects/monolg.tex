\descript{| An easy to use centralized Mongo db logging library}
\location{Python \textbullet{} MongoDB \textbullet{} Github Actions \textbullet{} Pytest}

A Simple easy to use centralized logging library for Python using MongoDB. Deployed at Python Packaging Index aka. PyPI at \href{https://pypi.org/project/monolg/}{\link{pypi.org/project/monolg}}. The library was written so that any Python developer who uses the default logging library can get started with Monolg as easily. With complete documentation and user guide hosted in Read the docs at \href{https://monolg.readthedocs.io/en/latest/}{\keyword{\link{monolg.readthedocs.io}}}.
